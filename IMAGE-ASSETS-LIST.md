## üé® **Image Specifications**

### **Recommended Image Formats:**

| Image Type | Format | Reason |
|------------|--------|--------|
| Logos | PNG or SVG | Transparency, scalability |
| Photos | JPG | Smaller file size |
| Graphics | PNG | Better quality for graphics |
| Icons | SVG | Scalable, small file size |

### **Recommended Image Dimensions:**

| Image Type | Dimensions | Aspect Ratio |
|------------|------------|-------------|
| Logo (Header) | 200x80px | 5:2 |
| Logo (Hero) | 400x160px | 5:2 |
| Hero Background | 1920x1080px | 16:9 |
| Thumbnails | 200x150px | 4:3 |
| Image Train | 400x300px | 4:3 |

### **Image Optimization Tips:**

1. **Compress images** before uploading (use TinyPNG, ImageOptim, etc.)
2. **Use WebP format** for better compression (with JPG/PNG fallback)
3. **Lazy load images** below the fold
4. **Use responsive images** with `srcset` for different screen sizes
5. **Add proper alt text** for accessibility and SEO
